The Artifacts feature surfaces files your agents create or capture during a session — screenshots, browser recordings, and documents — so you can review them without leaving the AI Monitor Pro UI.

<Callout title="Coming soon">
Full Artifacts documentation is coming soon. When published, this page will cover:

- **Screenshots** — images captured by browser automation agents (Antigravity and others), viewable inline with zoom.
- **Antigravity browser recordings** — sampled thumbnail strips from `browser_recordings` that let you scrub a visual replay of what the agent browsed.
- **Documents** — `task.md`, `plan.md`, `walkthrough.md`, and similar files written by the agent, rendered as formatted text in the browser.
- The Artifacts tab inside a session trace.
- How AI Monitor Pro locates artifact files relative to the session log.
</Callout>

Artifacts are already captured and linked from session traces — open any Antigravity session and click the **Artifacts** tab in the trace view to see what's available today.
